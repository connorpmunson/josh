# J.O.S.H. (Just One Smart Helper)

A local Python voice assistant designed for offline-first interaction.

## Features
- Wake word detection using Picovoice Porcupine
- Local speech-to-text with Whisper
- Local text-to-speech with Piper
- Alexa-style single command mode
- Extended dialogue mode
- Long-term memory storage
- Tool routing (time + web search)
- Silence-based audio recording

## Architecture
Core components:
- Wake word listener
- Audio recorder
- Whisper STT
- LLM brain
- Piper streaming TTS
- Tool router
- Memory system
